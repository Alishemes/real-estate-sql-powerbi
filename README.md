# 🏠 Real Estate Analytics | SQL & Power BI

## 📌 Project Overview
This project presents an end-to-end real estate data analysis workflow using SQL and Power BI. The analysis focuses on cleaning raw housing data, exploring key market trends, and building an interactive dashboard to visualize important business insights and KPIs.

---

## 🛠️ Tools & Technologies
- SQL
- Power BI
- Excel

---

## 📊 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- KPI Tracking & Analysis
- Interactive Dashboard Visualization
- Housing Market Insights

---

## 📈 Insights Generated
- Impact of crime rate on housing prices
- Relationship between number of rooms and property value
- Tax rate analysis
- Housing distribution by river proximity
